# Mnist, flask and Tensorflow

A flask app for playing with the Mnist dataset.

## Environment
<p>Anaconda Python 3.6</p>
Install librabries:

```
conda install requirements.txt
```

## Run the flask App
To run the app:
```
set FLASK_APP = app.py
flask run
```

Then connect to : http://127.0.0.1:5000/
